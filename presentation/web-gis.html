Web GIS ?
media / sistem yang digunakan untuk mengelola data geografis / spasial menjadi informasi
Data spasial?
data yang memiliki referensi ruang kebumian
Data spasial ada 2 jenis =
Raster dan Vector
Data raster?
data yang menampilkan sisi ruang bumi dalam bentuk pixel (gambar)
Contoh : GeoTIFF and JPEG2000
Isi data = Red, Green, and Blue
Data vector?
data yang menampilkan pola keruangan berupa titik, garis atau poligon
Contoh : Shapefile, GeoJson, KML dll
